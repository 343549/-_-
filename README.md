# Лабораторная работа №1 — Снижение размерности музыкальных данных

Проект посвящён анализу датасета `music_genre.csv`, содержащего ~50 тыс. треков и 18 признаков (категориальные и числовые). В ноутбуке `лаб1.ipynb` реализован полный конвейер предобработки и визуализации, позволяющий сравнить линейные и нелинейные методы уменьшения размерности.

## Структура решения

1. **Загрузка и первичный контроль** — проверка наличия входного файла и просмотр первых строк.
2. **Кодирование категориальных признаков** — `OrdinalEncoder` c сохранением CSV-маппингов в `analysis_outputs`.
3. **Отбор информативных признаков** — удаление нулевой вариации, выбор ТОП‑12 по дисперсии, оценка корреляций.
4. **PCA** — стандартизация, scree-график, вывод долей объяснённой дисперсии и визуализация PC1–PC2 (в том числе по жанрам).
5. **t-SNE** — подвыборка до 5000 треков, прогоны с `perplexity` 5/30/50 и цветовые карты по жанрам.
6. **Сравнение методов** — общий график PCA vs t-SNE и текстовое резюме (сохраняется в `analysis_outputs/analysis_summary.txt`).

Все промежуточные артефакты (графики, CSV, текстовые отчёты) сохраняются в папку `analysis_outputs`.

## Как воспроизвести

1. Установите зависимости (например, в виртуальной среде):
   ```bash
   pip install pandas numpy scikit-learn matplotlib seaborn
   ```
2. Скопируйте `music_genre.csv` в корень проекта (уже присутствует).
3. Запустите `лаб1.ipynb` в Jupyter/Colab последовательно по блокам (или выполните `Run All`). Результаты и комментарии появятся сразу под соответствующими ячейками.

## Основные выводы

- Сильная корреляция обнаружена только между `loudness` и `acousticness` (r ≈ -0.73), что позволяет использовать остальные признаки без явной избыточности.
- Первые пять компонент PCA объясняют ~54 % вариации, обеспечивая интерпретируемое линейное представление.
- t-SNE (на подвыборке 5000 строк) при `perplexity = 30` наглядно выявляет локальные жанровые кластеры, превосходя PCA по качеству визуального разделения.
- Финальный отчёт и наглядные сравнения находятся в `analysis_outputs`, что упрощает подготовку итогового отчёта или презентации.
